# noob-rust
Mes cinq premiers projets avec Rust.

###### Ma petite liste de commandes utiles
- **Cree un projet rust**
```cargo
cargo new nom-projet
```
- **Executer le projet :**
```cargo
cargo run
```
- **Compiler le projet**
```cargo
cargo build
```
- **Executer un seul fichier :**
```rustc
rustc nom-fichier.rs
```
- **Aficher la documentation**
```rustup 
 rustup doc
```
